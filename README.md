To understand cybersecurity fundamentals and create a virtual lab using kali Linux and Metasploitable2

1.VMware Workstation
2.Kali Linux
3.Metasploitable2
4.Nmap
5.Wireshark
# Activities performed
1.Installed Kali linux
2.Installed metasploitable2
3.Configured virtual machines.
4.Identified ip adderesses

7.Captured ICMP packets using Wireshark
# Results
Successfully created a cybersecurity lab environment and performed basic network analysis and scanning.
